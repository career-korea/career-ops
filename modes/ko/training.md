# 모드: training — 교육/자격증 가치 평가

강의, 자격증, 부트캠프 수강을 고려 중일 때 6개 차원으로 평가한다.

## 평가 6개 차원

| 차원 | 평가 내용 |
|------|----------|
| 목표 역할 정렬 | 목표 직무에 가까워지는가, 멀어지는가 |
| 채용 담당자 신호 | 이 자격증을 이력서에서 보면 어떻게 평가하는가 |
| 시간·노력 | 예상 기간 × 주당 시간 |
| 기회비용 | 그 시간에 다른 무엇을 할 수 있는가 |
| 리스크 | 내용이 오래됐는가? 브랜드가 약한가? 너무 기초적인가? |
| 포트폴리오 산출물 | 수료 후 보여줄 수 있는 결과물이 나오는가 |

## 한국 취업 시장 기준 우선순위

취준생·신입 기준:

1. **국가 공인 자격증** — 정보처리기사, ADsP, SQLD, 빅데이터분석기사 (서류 통과에 직접 영향)
2. **클라우드 자격증** — AWS SAA, GCP ACE, Azure AZ-900 (IT 직무 선호도 높음)
3. **AI/ML 실무 증명** — 캐글 경진대회 입상, Hugging Face 기여, 프로젝트 결과물
4. **코딩 역량** — 프로그래머스 레벨, 백준 티어 (개발직 채용 기준 활용)
5. **어학 점수** — 토익, OPIc, 토스 (대기업 공채 서류 기준 존재)

시니어 기준:
1. AI 프로덕션 역량 증명 (evals, observability, cost 최적화)
2. 아키텍처 레벨 자격증 (AWS SAP, GCP PCA)
3. 도메인 전문성 자격증

## 판정

- **할 것** → 4-12주 계획, 주차별 달성 목표 포함
- **하지 말 것** → 더 나은 대안과 근거 제시
- **기간 제한으로 할 것** (최대 N주) → 핵심 내용만 압축

# 모드: training — 교육/자격증 평가

후보자가 물어보는 각 강의/자격증에 대해 6개 차원을 평가한다:

| 차원 | 평가 내용 |
|-----------|------------|
| North Star 정합성 | 목표에 가까워지는가, 멀어지는가? |
| 리크루터 신호 | 채용 담당자(HM)가 CV에서 이것을 보면 어떻게 생각하는가? |
| 시간과 노력 | 주차 수 × 주당 시간 |
| 기회비용 | 그 시간 동안 못 하게 되는 것은? |
| 리스크 | 내용이 낡았나? 브랜드가 약한가? 너무 기초적인가? |
| 포트폴리오 산출물 | 증명 가능한 결과물이 나오는가? |

## 판정

- **하기(DO)** → 주별 산출물과 점수판이 있는 4~12주 계획
- **하지 않기(DON'T)** → 더 나은 대안 + 근거
- **타임박스로 하기(DO WITH TIMEBOX)** (최대 X주) → 핵심만 압축한 계획

## 우선순위

"프로덕션급 AI" 신뢰도를 높이는 교육을 우선한다:
1. LLM 평가(Evals)와 테스팅
2. 옵저버빌리티와 모니터링
3. 비용/안정성 트레이드오프
4. AI 거버넌스와 안전성
5. 엔터프라이즈 AI 아키텍처

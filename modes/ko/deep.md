# 모드: deep — 회사 심층 리서치

지원 회사에 대해 6개 축으로 구조화된 리서치를 실행한다. `cv.md`와 `config/profile.yml`을 읽어 후보자 맥락을 파악한 뒤 진행한다.

## 리서치 6개 축

```
## 심층 리서치: [회사명] — [역할]

### 1. AI/기술 전략
- 어떤 제품/서비스에 AI·ML이 쓰이는가?
- 기술 스택은? (모델, 인프라, 내부 도구)
- 기술 블로그나 GitHub 오픈소스가 있는가?
- AI 관련 논문, 발표, 특허가 있는가?

### 2. 최근 동향 (최근 6개월)
- AI/ML/제품 관련 주요 채용 이슈가 있는가?
- 인수합병, 파트너십, 투자 유치가 있었는가?
- 신제품 출시 또는 피벗이 있었는가?
- 구조조정, 임원 교체, 조직 변화가 있었는가?

### 3. 개발 문화
- 배포 주기, CI/CD 방식은?
- 사용 언어/프레임워크는? (채용공고, 기술 블로그 기준)
- 원격·하이브리드·오피스 정책은?
- 포괄임금, 야근 문화, 워라밸 평판은? (잡플래닛, 블라인드 기준)
- 내부 개발자가 가장 많이 언급하는 불만/칭찬은?

### 4. 예상 기술 과제
- 어떤 스케일 문제를 겪고 있는가?
- 비용, 지연, 안정성 이슈가 있는가?
- 인프라·모델·플랫폼 마이그레이션 진행 중인가?
- 리뷰에서 자주 언급되는 페인포인트는?

### 5. 경쟁사 및 포지셔닝
- 주요 경쟁사는?
- 차별화 포인트(해자)는?
- 시장에서의 위치와 성장 방향은?

### 6. 후보자 각도
cv.md와 profile.yml 기반으로:
- 이 팀에서 내가 제공할 수 있는 고유한 가치는?
- 가장 관련성 높은 프로젝트/성과는?
- 면접에서 어떤 스토리를 중심으로 말할 것인가?
```

## 한국 리서치 소스

| 목적 | 소스 |
|------|------|
| 연봉/보상 | 잡플래닛, 크레딧잡, 블라인드, 원티드 인사이트, Levels.fyi |
| 직원 리뷰/문화 | 잡플래닛, 블라인드, 리멤버 |
| 기술 스택 | 기술 블로그, GitHub, 채용공고, 링크드인 |
| 뉴스/동향 | 테크크런치 코리아, 블로터, 디지털타임스, 회사 공식 블로그 |
| 재무/투자 | 더브이씨, 혁신의숲, DART(금융감독원 공시) |

## 출력 형식

각 섹션을 채워서 제공한다. 정보가 없는 항목은 "확인 불가 — 추가 조사 필요"로 표시하고 추측하지 않는다. 모든 수치와 주장에 출처를 표기한다.

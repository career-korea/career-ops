# 모드: ofertas -- 여러 공고 비교

여러 공고를 한국 + 글로벌 리모트 기준으로 비교한다.

| 차원 | 가중치 | 1-5 기준 |
|---|---:|---|
| 목표 역할 적합도 | 25% | 목표 아키타입과의 일치 |
| CV 매칭 | 15% | JD 요구사항과 실제 근거 매칭 |
| 레벨 | 15% | 후보자 자연 레벨과 역할 범위 |
| 보상 | 10% | 후보자 목표 및 시장 기준 대비 |
| 성장성 | 10% | 다음 레벨/학습/책임 범위 |
| 근무 품질 | 5% | 원격/하이브리드/시간대/출근 부담 |
| 회사 신뢰도 | 5% | 평판, 재무/채용 안정성 |
| 기술 스택 | 5% | 현대성, 후보자 방향성과의 일치 |
| 채용 속도 | 5% | 프로세스 명확성, 타임라인 |
| 문화 신호 | 5% | 빌더 문화, 자율성, 관료성 |

한국 공고는 다음을 별도 메모한다:
- 연봉 공개 여부
- 포괄임금/고정 OT
- 근무지/재택 정책
- 수습기간/계약형태

글로벌 리모트 공고는 다음을 별도 메모한다:
- 시간대 겹침
- 계약/EOR 가능성
- 통화/세금 리스크
- 지리 제한

최종 출력:
- 순위표
- 각 공고의 강점/리스크
- 지원 우선순위
- 지금 지원하지 말아야 할 공고

# 모드: ofertas — 다중 채용공고 비교

가중치가 부여된 10개 차원의 점수 매트릭스로 비교한다:

| 차원 | 가중치 | 1~5 기준 |
|-----------|------|----------------|
| North Star 정합성 | 25% | 5=정확히 목표 역할, 1=무관 |
| CV 매칭 | 15% | 5=90% 이상 일치, 1=40% 미만 |
| 직급(시니어 이상) | 15% | 5=staff+, 4=senior, 3=mid-senior, 2=mid, 1=junior |
| 예상 보상 | 10% | 5=상위 25%, 1=시장가 이하 |
| 성장 경로 | 10% | 5=다음 단계로 명확한 경로, 1=막다른 길 |
| 원격근무 품질 | 5% | 5=완전 원격 비동기, 1=온사이트만 |
| 회사 평판 | 5% | 5=최고 고용주, 1=위험 신호 |
| 기술 스택 현대성 | 5% | 5=최첨단 AI/ML, 1=레거시 |
| 오퍼까지 속도 | 5% | 5=빠른 프로세스, 1=6개월 이상 |
| 문화 신호 | 5% | 5=빌더 문화, 1=관료적 |

각 공고에 대해: 차원별 점수와 가중 합산 총점을 매긴다.
최종 순위 + 오퍼까지 걸리는 시간을 고려한 추천을 제시한다.
모든 리포트의 헤더에는 **URL:** 과 **Legitimacy:** 필드를 반드시 포함한다.

컨텍스트에 채용공고가 없으면 사용자에게 요청한다. 텍스트, URL, 또는 이미 트래커에서 평가한 공고 참조 모두 가능하다.
